Installing the OCI/Thick Driver for Oracle SQL Developer

  1. Download Oracle Client Software

    Download the appropriate Oracle Client software from the Oracle website. Select the version that matches your Oracle Database version and your operating system. The "Instant Client" is a lightweight version that includes the basics for OCI, while the full Oracle Client includes additional tools and libraries.

  2. Install Oracle Client

    Install the Oracle Client software on your system. The process will depend on the version and your operating system, but generally involves unzipping the downloaded file and running an installer.

  3. Set the ORACLE_HOME Environment Variable

    The ORACLE_HOME environment variable should be set to the directory where you installed the Oracle Client.

  4. Configure SQL Developer to Use the OCI/Thick Driver

    1. Open SQL Developer.
    2. Go to Tools -> Preferences.
    3. In the Preferences dialog, go to Database -> Advanced.
    4. Under the "Use OCI/Thick driver" section, select the checkbox that says "Use OCI/Thick driver".
    5. In the "OCI Library" field, enter the path to the oci.dll (on Windows) or libclntsh.so (on Linux or macOS) file in your Oracle Client installation directory. Typically, this file is in the "bin" subdirectory of the ORACLE_HOME directory.
    6. Click "OK" to close the Preferences dialog.

  5. Restart SQL Developer

    Changes will take effect after you restart SQL Developer.

Remember to check Oracle's official documentation for the most accurate and up-to-date information.
